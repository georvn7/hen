
PROJECT DESCRIPTION


Simple C Compiler (C17 standard revision)

Command Line Interface:
scc [options] source_file
Options:
  -o <file path>: Output assembly source file path
  -v: Enable verbose error reporting
Example: scc -o "~/Documents/output.s" input.c

Supported Keywords: 
break, case, const, continue, default, do, else, enum, extern, for, goto, if, return, sizeof, static, struct, switch, union, while

Supported Operators:
   - Arithmetic: +, -, *, /, %, ++, --
   - Relational: ==, !=, >, <, >=, <=
   - Logical: &&, ||, !
   - Bitwise: &, |, ^, ~, <<, >>
   - Assignment: =, +=, -=, *=, /=, %=, <<=, >>=, &=, ^=, |=
   - Special: sizeof, & (address), * (dereference), ->, ., ?:, ,

Supported Types:

char - 8 bits signed value
int - 32 bits signed value
short - 16 bits signed value
float - 32 bits floating point number

By default, all types are signed. Unsigned versions available with 'unsigned' modifier.

Custom data type definitions are supported by struct, enum and union features.

Error Handling:
Normal mode (default): Provides sufficient error details.
Verbose mode (-v): Offers comprehensive error reports.

Error handling in normal and verbose mode should report errors appropriately.

Code Generation:
- Target: ARM architecture 64-bit assembly source (ASCII format)
- The generated output assembly source must be able to compile with clang. Example: clang --target=aarch64 -c source.s -o source.o
- Object files, compiled with clang, from the generated by scc assembly sources must be able to link with other object files generated with clang toolchain and the C standard library.
- Single-threaded compiler
- No linking support (assembly output only)
- IR support optional (only if simplifies the implementation)
- Do not generate any kind of meta data or debug information in addition to the assembly source.

Implementation Instructions:
- DO NOT Implement preprocessor. Assume fully preprocessed input C source files (ASCII format)
- Support only the features listed above
- Focus on basic functionality
- ASCII input files only
- Maintain simplicity to ensure functionality.
